AI 라이선스 대출 시스템 개발 대화 기록
===========================================

📅 날짜: 2025년 7월 29일
🎯 목표: AI 라이선스 대출 시스템 구축

## 🔧 개발 과정

### 1. 프로젝트 초기 설정
- 프로젝트 디렉토리 생성: ai-license-system
- 서버(Backend) 및 클라이언트(Frontend) 구조 설정

### 2. 기술 스택
**Backend:**
- Node.js + Express.js
- SQLite3 데이터베이스
- JWT 인증
- bcryptjs 암호화

**Frontend:**
- React.js
- React Router
- Axios HTTP 클라이언트
- 반응형 CSS 디자인

### 3. 주요 기능 구현
✅ 기관별 라이선스 관리
✅ 라이선스 대출/반납 시스템
✅ 대출 기간 자동 설정
✅ 대출 내역 조회
✅ JWT 기반 보안 인증
✅ 사용자 등록/로그인

### 4. 데이터베이스 구조
- organizations (기관)
- users (사용자)
- ai_licenses (AI 라이선스)
- license_loans (라이선스 대출)

### 5. 설치 및 실행 과정
- Node.js 설치 (v22.17.1)
- 환경 변수 설정 문제 해결
- PowerShell 실행 정책 설정
- 의존성 설치 완료

### 6. 현재 상태
✅ 서버: 데이터베이스 초기화 완료, 포트 5000에서 실행
⚠️ 클라이언트: React 앱 컴파일 완료, 서버 연결 대기 중

## 🚀 다음 단계

```bash

```

```bash

```

## 📝 참고 사항
- Node.js 설치 시 "Add to PATH" 옵션 필수
- PowerShell에서 환경 변수 임시 설정: $env:Path += ";C:\Program Files\nodejs"
- 실행 정책 설정: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
